# Open Access: Background and Tools for Early Career Researchers in Social Sciences

**Philippe Joly, Humboldt-Universität zu Berlin & WZB Berlin Social Science Center**

_July 2018_

This repository contains the material of my workshop on open access that was presented at the Berlin Summer School in Social Sciences on July 18, 2018. This project was prepared as part of the [Fellowship Freies Wissen](https://wikimedia.de/wiki/BildungWissenschaftKultur/Fellowprogramm) sponsored by Wikimedia Deutschland, the Stifterverband, and the VolkswagenStiftung.
